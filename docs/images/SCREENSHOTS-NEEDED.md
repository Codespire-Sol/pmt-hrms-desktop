# Screenshots to capture (drop them in this folder)

Save each as a **PNG** with the **exact filename** below (lowercase, hyphens).
Tips for a clean, enterprise look:
- Maximize the app window first; capture the **whole window** (Win: `Alt + PrtScn`, or Snipping Tool `Win + Shift + S`).
- Use **realistic dummy data** (a few projects / employees) — not empty screens.
- Avoid showing real personal data or the admin password.
- Try to keep them a consistent size (e.g. ~1400×900).
